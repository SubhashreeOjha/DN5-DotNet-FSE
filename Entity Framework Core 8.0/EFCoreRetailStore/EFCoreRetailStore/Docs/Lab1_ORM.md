# Lab 1 - Understanding ORM (Object Relational Mapping)

## Objective
Understand how Object Relational Mapping (ORM) works and how Entity Framework Core simplifies database operations.

## What is ORM?
ORM (Object Relational Mapping) is a technique that maps database tables to C# classes and table rows to objects.

Instead of writing SQL queries manually, developers interact with C# objects and Entity Framework Core generates SQL automatically.

## Advantages of ORM
- Reduces boilerplate SQL code
- Improves developer productivity
- Strongly typed queries
- Easier maintenance
- Database independence
- Automatic relationship mapping

## Entity Framework Core
Entity Framework Core (EF Core) is Microsoft's lightweight, cross-platform ORM for .NET applications.

It supports:
- CRUD Operations
- LINQ Queries
- Migrations
- Change Tracking
- Relationships
- Code First Development

## Outcome
Successfully understood ORM concepts and the role of Entity Framework Core in .NET applications.
